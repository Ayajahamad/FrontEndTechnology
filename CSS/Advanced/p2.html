CSS CURSOR..!

CSS cursor property used to difine cursor type (i.e. mouse pointer) when the mouse moves over a certain area or, over a link on the page.

EXAMPLE..!

THIS IS FIRST PARAGRAPH..! (AUTO)

THIS IS SECOND PARAGRAPH..! (DEFAULT)

THIS IS THIRD PARAGRAPH..! (NONE)

THIS IS FOURTH PARAGRAPH..! (CONTEXT-MENU)

THIS IS FIFTH PARAGRAPH..! (HELP)

THIS IS SIXTH PARAGRAPH..! (POINTER)

THIS IS SEVENTH PARAGRAPH..! (PROGRESS)

THIS IS EIGHTH PARAGRAPH..!(WAIT)

THIS IS NINENTH PARAGRAPH..! (CELL)

THIS IS TENTH PARAGRAPH..! (CROSSHAIR)

THIS IS ELEVENTH PARAGRAPH..! (TEXT)

THIS IS TWELTH PARAGRAPH..! (VERTICAL-TEXT)

THIS IS THIRTEENTH PARAGRAPH..! (ALIAS)

THIS IS FOURTEENTH PARAGRAPH..! (COPY)

THIS IS FIFTEENTH PARAGRAPH..! (MOVE)
