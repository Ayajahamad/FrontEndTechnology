AJAX

Asynchronous Javascript add XML, or Ajax, is not a technology in itself, but rather an approach to using a number of existing technologies together, including HTML or XHTML, CSS, Javascript, DOM, XML, XSLT, and most importantly the XMLHttpRequest object.
